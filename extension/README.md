# InstructionsCrafter — Chrome Extension

This directory contains the Chrome extension portion of [InstructionsCrafter](../README.md), an internal tool that turns a narrated screen recording into a polished Microsoft Word step-by-step instruction document.

For installation, usage, and deployment instructions, see the [root README](../README.md).
